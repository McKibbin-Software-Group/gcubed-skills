## Project Memory

Use `$project-memory` to create or refresh repo memory docs. Keep repo
`AGENTS.md` to repo-specific guardrails and pointers; durable project truth
belongs in `docs/`, temporary handoff or investigation notes in `docs/ai/`.
Keep historical status/information only as long as necessary. Regularly remove
historical cruft or move it to a history document if necessary.
